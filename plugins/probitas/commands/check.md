## Workflow

1. **Format** - Run `probitas fmt`
2. **Lint** - Run `probitas lint`
3. **Check** - Run `probitas check`
4. **Fix** - If issues found, fix them and repeat

```bash
probitas fmt
probitas lint
probitas check
```

Fix issues iteratively until all pass.
